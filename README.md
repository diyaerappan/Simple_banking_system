# Simple Banking System

## Overview

Simple Banking System is a beginner Python project that simulates basic banking operations. Users can check their balance, deposit money, and withdraw money through a menu-driven interface.

## Features

- Check account balance
- Deposit money
- Withdraw money
- Balance validation
- Menu-driven program

## Technologies Used

- Python

## Concepts Covered

- Variables
- Data Types
- Loops
- Conditional Statements
- User Input
